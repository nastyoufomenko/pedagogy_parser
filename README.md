# Парсер карточек статей с главной страницы Педсовета

Учебный проект по парсингу HTML с использованием библиотеки **BeautifulSoup**.

Скрипт `parse_pedagogy.py` извлекает с главной страницы сайта [pedsovet.org](https://pedsovet.org/) «малые карточки» статей (небольшие прямоугольные виджеты) и собирает из них:

- наименование статьи (заголовок/подпись на карточке);
- ссылку на полную статью (абсолютный URL).

Результат можно вывести в консоль и при необходимости сохранить в JSON-файл.

## Что именно делает программа

1. Получает HTML главной страницы сайта:
   - либо напрямую с сайта через `requests`;
   - либо из заранее сохранённого HTML-файла (если передан параметр `--file`).
2. С помощью **BeautifulSoup** находит на странице блоки-карточки со статьями.
3. Для каждой карточки извлекает:
   - текст заголовка (наименование статьи);
   - ссылку на полную статью.
4. Обрабатывает относительные ссылки вида `/article/...`, дополняя их базовым доменом `https://pedsovet.org`.
5. Складывает данные в список словарей вида:

   ```python
   {
       "title": "Название статьи",
       "url": "https://pedsovet.org/article/..."
   }
